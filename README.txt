Current Source Code Finds the named entities from the wikipedia data based on the three categories.

Person
Location
Organisation

It is based on the context based approach as stated in the the paper: **https://www.idi.ntnu.no/~noervaag/papers/AINA2010.pdf
It matches the pattern and then decides which entity it is.

How to run:
1. Run the script run.sh as 
	bash run.sh {path to xml file}
